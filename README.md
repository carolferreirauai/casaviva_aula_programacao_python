### 👩‍🏫 Sobre o Repositório
Este espaço é dedicado ao armazenamento e compartilhamento de todo o material utilizado nas aulas de programação em Python ministradas na **Casa Viva**. 

Aqui, os alunos podem encontrar os códigos feitos em sala, listas de exercícios e exemplos práticos para auxiliar no estudo e revisão do conteúdo.

**Instrutora:** Caroline Ferreira de Souza

### 🚀 Tecnologias e Ferramentas
* **Linguagem:** Python 3.x

### 💡 Como utilizar este repositório
1. Faça o clone do projeto para a sua máquina local:
   `git clone https://github.com/SEU-USUARIO/NOME-DO-REPOSITORIO.git`
2. Navegue pelas pastas para encontrar os materiais separados por aulas ou temas.
3. Pratique rodando os scripts no seu próprio ambiente de desenvolvimento!
